# Pre-processing
These steps are intended to perform the demultiplexing of the data and apply all necessary quality filters prior to sequence alignment. This includes the following steps:

   * [1.1. Demultiplex](1.1_demultiplex) - Using bclconvert
   * [1.2. Created FastQC report](1.2_fastcq) - Using FastQC
   * [1.3. Trimming](1.3_trimming) - Using TrimGalore
